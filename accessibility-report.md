#Accessibility Report

##Issues Found

1. Link text could be more descriptive.
2. Accessibility score was not perfect and required review.

##Fixes Applied

1. Added "lang="en"" to the "<html>" tag.
2. Added alt text to images.
3. Verified proper heading hierarchy ("h1" → "h2").
4. Updated link text to be more descriptive.
5. Checked that all form inputs have associated labels.

Accessibility Score: 97/100

##Conclusion

The website was audited using Chrome Lighthouse. Most accessibility requirements were met, resulting in a score of 97/100.
